Web documentaire

En groupe de 4, nous avons réalisé un web documentaire sur la fête du brie de Meaux, Le Brie Happy.


Dans un premier temps, nous avons tourné les images sur place pendant l'évènement, puis nous avons réalisé le montage avec le logiciel Adobe Premiere Pro.
Ensuite, nous avons développé le site pour y inclure les vidéos et les articles. Nous sommes très fiers de ce web documentaire car il nous a plongé dans un réel contexte de tournage pour un documentaire.
